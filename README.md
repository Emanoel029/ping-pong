<h1>Ping-Pong</h1>
<p>Projeto criado em Javascript PURO desenhado campo, polinha e raquetes com JS</p>
<p>Game feito para jogar com computador! A raquete acompanha o mouse do jogaro no eixo Y. O PC acompanha a polinha que é incrementada um pouco de dificudade para o PC para não ficar em difício para o jogador consegui pontuar. 
  O game construidos orientado a objeto, todos os elementos do jogo eles vão coresponder a ações e propriedades. Exemplo: A bolinha tem uma propriedade x e y também pode ter o médoto desenhar e mover.

Confira nosso joguinho no link: https://emanoel029.github.io/ping-pong/</p>
